# Projects

## 1. User Directory Dashboard

A dynamic user management interface that fetches user data from an API and displays it in interactive cards.

### Features

✅ Fetch users from JSONPlaceholder API  
✅ Display user information dynamically  
✅ Search users by name or email  
✅ Sort users A-Z and Z-A  
✅ Filter users by city  
✅ Loading state handling  
✅ Error handling  

---

## 2. Weather Dashboard 🌦️

A real-time weather application using the OpenWeatherMap API.

Users can search for any city and receive current weather information.

### Features

✅ Search weather by city  
✅ Real-time API data  
✅ Temperature display  
✅ Celsius/Fahrenheit conversion  
✅ Weather icons  
✅ Humidity information  
✅ Wind speed information  
✅ Error handling for invalid cities  
✅ Loading state  
✅ Enter key search support
