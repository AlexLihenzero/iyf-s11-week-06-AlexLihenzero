# Week 6 - Asynchronous JavaScript & APIs

## Project: User Directory Dashboard

A JavaScript application that fetches user data from an external API and displays it dynamically in the browser.

This project demonstrates modern asynchronous JavaScript concepts including Fetch API, Promises, Async/Await, API error handling, and DOM manipulation.

---

## Features

### API Integration
- Fetch users from JSONPlaceholder API
- Handle asynchronous requests using Async/Await
- Convert API responses from JSON format

### User Interface
- Display users dynamically using JavaScript
- Generate user cards from API data
- Loading state handling
- Error message handling

### User Management Features
- Search users by name or email
- Sort users alphabetically:
  - A-Z
  - Z-A
- Filter users by city

---

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)
- Fetch API
- JSONPlaceholder API
